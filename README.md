# Тестовое задание: Автоматизация SauceDemo

## Описание
Проект содержит автотесты для сайта [SauceDemo](https://www.saucedemo.com/). 
Реализовано на **Python 3.10** с использованием **Playwright** и паттерна **Page Object Model**.

## Как запустить
Для запуска тестов в изолированной среде:
docker build -t my-tests .
docker run --rm my-tests


## Результаты
Все 5 тестов проходят успешно. Скриншот успешного прогона в Docker находится в корне проекта.
